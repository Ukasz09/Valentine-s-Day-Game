# Game controls
- Arrows left/right: flame shot
- Space: bomb
- WSAD: sprite control

# For Devs
Used Designs Patterns:
- Builder
- Strategy

# JAR
If there is a problem with running, try to open it by console with command:
"java -jar ValentinesGame.jar"
